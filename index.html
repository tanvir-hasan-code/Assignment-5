Question-1: What are the different ways to select an element in the DOM?

There are several ways to select an element in the DOM. One way is to select an element by its unique ID. Another method is to select elements based on their class name or tag name. Additionally, elements can be selected using CSS selectors to target specific elements in the document. Some methods return a single element, while others return a collection of matching elements.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The main difference between innerHTML, innerText, and textContent is how they handle and retrieve content from an element. innerHTML returns or sets the HTML content inside an element, meaning it processes and renders any HTML tags within it. On the other hand, innerText retrieves or sets only the visible text of an element, ignoring hidden elements and extra spaces. textContent, however, gets or sets the full text of an element, including hidden text, but does not process HTML tags.

Question-3: What is event delegation in the DOM?

Instead of attaching individual event listeners to each element, event delegation involves attaching a single event listener to a parent element that covers all the target elements. When an event is triggered on a target element, it bubbles up to the parent, where the event listener is listening. From there, the event handler can determine which child element triggered the event.

Question-4: What is event bubbling in the DOM?

Event Bubbling is a process in the DOM where an event occurs on an element, and then it bubbles up (or propagates) to its parent elements and further up the DOM tree until it reaches the root (document). In other words, when an event happens, it first triggers on the element where the event occurred, and then it "bubbles up" through the parent elements, one by one, until it reaches the document.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element in JavaScript, use document.createElement('tagName') to generate a new element. Add it to the DOM using methods like appendChild() for appending or insertBefore() for inserting before a specific element. To remove an element, use remove() or removeChild(). These methods allow you to dynamically manage DOM elements by creating, adding, and removing them as needed.
